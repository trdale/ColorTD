HTML5 Canvas Tower Defense Game.

Game can be running using AG-start.html on local drive.

Also found [ADDRESS]


Game is played by placing towers clicking '1' on the keyboard then moving the mouse to where you wish to place them and clicking on the mouse. '2', '3', and '4' will allow you to click on a tower to insert the corresponding colored gem.

Gem Costs:

1st Gem 200
2nd Gem 400
3rd Gem 600

Gem Effects:

Red Gem: Damage Increase
Blue Gem: Increases max targets
Green Gem: Increases max range

The game is won when 10 waves are completed.
